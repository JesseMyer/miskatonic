# MISKATONIC

> The institution that governs access to forbidden architectural knowledge.

Miskatonic is a governed AI platform for enterprise architecture practice.

## Subsystems

| Subsystem | Name | Function |
|---|---|---|
| Prompt Library | **Vril** | The activating invocations |
| Pattern Corpus | **Ossuary** | The governed repository of preserved patterns |
| ZUI Framework | **Tenebrous** | The metamodel descent engine |
| Governance Engine | **Bureau** | The adjudication layer |
| Roadmap Engine | **Pelagic** | The governed passage between states |
| Annotation Harness | **Grimoire** | The record of accumulated judgment |

## The Governing Sentence

Vril invokes Ossuary. Tenebrous renders the view. Bureau scores the architecture. Pelagic charts the passage. Grimoire records what was learned.
